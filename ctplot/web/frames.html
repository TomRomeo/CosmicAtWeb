 Wähle bitte Polarstern aus. select[name=experiment0] Polarstern Diagramm erstellen: Datensatz Alle Datensätze, die dir für das jeweilige Experiment zur Verfügung stehen, sind hier aufgelistet. Datensätze werden folgendermaßen gekennzeichnet: ZEITRAUM_EXPERIMENT_ANMERKUNGEN Wähle nun den Datensatz von 2014. select[name=s0] Polarstern/2014_PS-nm-my.h5:/raw/PS_mu_nm_data Diagramm erstellen: Darstellungsform Nun kann ein geeignetes Diagramm für die Darstellung der Daten ausgesucht werden. Man unterscheidet vier verschiedene Diagrammtypen:
  1. 1D-Histogramm: Ermöglicht die grafische Darstellung einer einzelnen Variablen. Auf der x-Achse wird die Variable, auf der y-Achse deren Häufigkeit dargestellt.
  2. xy-Diagramm: Damit kann die Abhängigkeit einer Variablen von einer anderen dargestellt werden, so wie man es aus der Schule kennt.
  3. 2D-Histogram: Damit kann die Abhängigkeit einer Variablen von einer anderen dargestellt werden (genau wie beim xy-Diagramm), wobei zusätzlich die Häufigkeit durch eine Farbskala gekennzeichnet ist.
  4. Karte: Darstellung der geographischen Koordinaten der Polarstern. Optional kann eine weitere Variable in Abhängigkeit der Koordinaten farblich skaliert dargestellt werden.
    Hinweis: Bei dieser Darstellungsform sind nur die Variablen Longitude und Latitude als x- bzw. y-Variable sinnvoll.
  5. Profil: Dieser Diagramtyp wird hauptsächlich im Expertenmodus verwendet, wird zur Vollständigkeit aber aufgelistet. Informationen dazu findest du hier.
Wir werden für unser Beispiel ein xy-Diagramm verwenden. select[name=m0] xy Diagramm erstellen: X-Variable Hier kann ausgewählt werden, welche gemessene Größe auf der x-Achse abgetragen werden soll.

Beschreibungen und Erklärungen zu den jeweiligen Variablenarten findest du hier.
